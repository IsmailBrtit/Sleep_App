# Sleep App – Installation Instructions

##  Clone the Project

```bash
git clone https://github.com/IsmailBrtit/Sleep_App


Notifications & DND :
Aller dans Paramètres > Applications > Sleep App > Notifications

✅ Autoriser toutes les notifications

Aller dans Paramètres > Sons et notifications > Accès spécial > Ne pas déranger

✅ Activer l'accès pour Sleep App

Alarme exacte (Exact Alarms) :
Aller dans Paramètres > Accès spécial > Alarme et rappels

✅ Autoriser Sleep App

Batterie :
Aller dans Paramètres > Applications > Sleep App > Batterie

✅ Sélectionner "Utilisation illimitée" (ou "Non restreinte")
