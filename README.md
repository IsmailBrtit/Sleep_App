# Sleep App – Installation Instructions

##  Clone the Project

```bash
git clone https://github.com/IsmailBrtit/Sleep_App

L’application nécessite Android 13 (API 33) ou une version supérieure pour fonctionner correctement.
Elle a été testée sur un émulateur Pixel 6a – API 36 (Android 15 Preview).

##  Configuration requise

1. Ouvrir le projet avec Android Studio.
2. Générer automatiquement `local.properties` .
3. Le fichier google-services.json est déjà inclus dans /app
4. Cliquer sur `Sync Project with Gradle Files` dans Android Studio.
5. Lancer l'application.



Notifications & DND :
Aller dans Paramètres > Applications > Sleep App > Notifications

✅ Autoriser toutes les notifications

Aller dans Paramètres > Sons et notifications > Accès spécial > Ne pas déranger

✅ Activer l'accès pour Sleep App

Alarme exacte (Exact Alarms) :
Aller dans Paramètres > Accès spécial > Alarme et rappels

✅ Autoriser Sleep App

Batterie :
Aller dans Paramètres > Applications > Sleep App > Batterie

✅ Sélectionner "Utilisation illimitée" (ou "Non restreinte")
